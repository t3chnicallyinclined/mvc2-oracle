How to read it

Each character has a color. Top: its on-screen box. Bottom: its 0x5A4 struct as a byte grid. Bytes flare in the object color the frame they change ("owned + changed"). The dim striped band is the engine pointer cluster (+0x154..184) — it churns every frame, so it's demoted as noise. Hover either view to link them.
